enum cy_en_usbpd_poll_vsys_t

List of values for poll for vsys variable.

Bit 0 is used for enable the vsys polling. Bit 1 is used for starting the delay timer in the task. Bit 2 is used for stop re-starting the timer once it is started.

Enumerator
CY_USBPD_POLL_VSYS_DO_NOT_POLL 

Explicit VSYS switching happened.

Do not poll for VSYS change.

CY_USBPD_POLL_VSYS_IDLE 

Idle and poll for VSYS.

CY_USBPD_POLL_VSYS_INT_UP 

Interrupt happened while in explicit switch.

Hence do not start the delay timer.

CY_USBPD_POLL_VSYS_START_TIMER 

Interrupt happened while in Idle state.

start the delay timer.

CY_USBPD_POLL_VSYS_STOP_TIMER 

Timer is started.

stop re-starting it again..
