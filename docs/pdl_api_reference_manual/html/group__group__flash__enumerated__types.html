enum cy_en_flashdrv_status_t

Status of the Flash driver.

Enumerator
CY_FLASH_DRV_SUCCESS 

Success.

CY_FLASH_DRV_INVALID_PROT 

Invalid device protection state.

CY_FLASH_DRV_INVALID_FLASH_ADDR 

Invalid Flash address.

CY_FLASH_DRV_ROW_PROTECTED 

Row is write protected.

CY_FLASH_DRV_INVALID_INPUT_PARAMETERS 

Input parameters passed to Flash API are not valid.

CY_FLASH_DRV_DEV_NOT_SUPPORTED 

API not supported for the selected device.

CY_FLASH_DRV_ERROR_UNKNOWN 

Unknown error code.

CY_FLASH_DRV_PROGRESS_NO_ERROR 

Command in progress; no error.

Status returned by Cy_Flash_IsOperationComplete() when a Flash operation is in progress

CY_FLASH_DRV_OPERATION_STARTED 

Flash operation is successfully initiated.

Status returned when Cy_Flash_StartWrite() succeeds

CY_FLASH_DRV_PENDING_RESUME 

Flash is under operation.

Status is returned when flash non-blocking operation is complete and waiting for Cy_Flash_ResumeWrite() API call
