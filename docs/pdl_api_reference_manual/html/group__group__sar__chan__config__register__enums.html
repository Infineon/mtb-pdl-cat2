enum cy_en_sar_chan_config_port_pin_addr_t

Configure the input mode of the channel.

  • Single ended channel: the cy_en_sar_ctrl_neg_sel_t selection in the group_sar_init_struct_ctrl register determines what drives the Vminus pin
  • Differential: Vplus and Vminus are a pair. Bit 0 of cy_en_sar_chan_config_port_pin_addr_t is ignored and considered to be 0. In other words, cy_en_sar_chan_config_port_pin_addr_t points to the even pin of a pin pair. The even pin is connected to Vplus and the next odd pin is connected to Vminus. Configure address of the port and the pin connected to the Vplus terminal of the SARADC See the SARMUX and SARSEQ section for more guidance.
