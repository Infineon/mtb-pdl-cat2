uint8_t cy_stc_dsadc_dchan_result_tag_t::sampleCount

6-bit sequential counter for valid outputs, incremeted on each valid output.

When averaging is enabled it increments only when the averaged value becomes valid. It resets whenever the digital channel or the whole PACSS block is disabled.
