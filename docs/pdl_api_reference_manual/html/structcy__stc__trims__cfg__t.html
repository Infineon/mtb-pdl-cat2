uint16_t cy_stc_trims_cfg_t::offset_20csa

Variables are created to use dynamic CC/CF gain setting at 3A switchover.

For 3A application, using fixed gain of 80 across 1A to 3A. For 5A application, using fixed gain of 60 across 1A to 5A. 20CSA current sense offset in 10uV with gain = 1. This offset shall be treated as 20CSA trim. This trim has to be subtracted while sampling 20CSA output. And added, while setting reference to 20CSA trigger.
